# The Order Remembererer

A tool to help you and a group of people remember each other's takeaway orders

## Models

### `Venue`

The place to which an `Order` pertains

### `Cuisine`

What type of food a `Venue` serves

### `Order`

What a `User` gets from a `Venue`

### `Team`

A group of people - a `User` can belong to many `Team`s
